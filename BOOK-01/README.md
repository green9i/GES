# BOOK-01

# Constitution

---

## Purpose

BOOK-01 определяет фундаментальные инженерные основы GENESIS Engineering Standard.

Книга содержит документы, описывающие философию, терминологию, принципы и базовые правила построения GES.

---

## Scope

В книгу входят:

- Constitution;
- Principles;
- Glossary;
- базовые инженерные модели.

Не входят:

- архитектурные решения (ADR);
- инженерные стандарты;
- рабочие регламенты;
- эксплуатационная документация.
