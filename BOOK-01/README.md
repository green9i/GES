# Книга 1. Основы GES

## Назначение

Книга определяет фундаментальные основы GENESIS Engineering Standard.

Она устанавливает:

- философию инженерной системы;
- конституцию GES;
- базовые инженерные принципы;
- терминологию;
- модель инженерной деятельности;
- основные правила построения стандарта.

---

## Принцип

Все инженерные решения GES должны соответствовать фундаментальным принципам, определенным в данной книге.

Документы BOOK-01 являются основой для всех остальных книг и обладают наивысшим архитектурным приоритетом.
