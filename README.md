# GENESIS Engineering Standard

## Назначение

GENESIS Engineering Standard (GES) — инженерная система управления знаниями, архитектурой и развитием компании GENESIS Group.

GES обеспечивает сохранение, развитие и передачу инженерных знаний независимо от:

- программного обеспечения;
- организационной структуры;
- сотрудников;
- внешних инструментов.
