# ST-007

# Связи между инженерными записями

## Назначение

Определяет правила формирования взаимосвязей между объектами GES.

GES является системой связанных инженерных записей.

---

# Допустимые типы связей

## Основано на

Используется, когда запись создана на основании другой записи.

Пример:


ADR-001

основано на

RS-001


---

## Изменяет

Используется при изменении существующей записи.

Пример:


ADR-002

изменяет

ST-003


---

## Заменяет

Используется при выпуске новой версии.

Пример:


ST-004 v2.0

заменяет

ST-004 v1.0


---

## Отменяет

Используется при признании решения недействительным.

---

## Использует

Используется, если объект применяется другим объектом.

---

## Связано с

Используется только при наличии смысловой зависимости.

---

# Ограничение

Связь должна объяснять причину существования.

Запрещаются связи:

- только для увеличения количества ссылок;
- без понятного назначения;
- созданные автоматически без проверки.
