# ST-008

# Единственный источник истины

## Назначение

Определяет правила хранения инженерных записей.

---

# Принцип

Каждая инженерная запись существует в единственном мастер-экземпляре.

---

# Модель


Инженерная запись

↓

Мастер-файл Markdown

↓

Представления

DOCX

PDF

HTML

Wiki

---

# Правила

Запрещается:

- создавать независимые копии;
- изменять экспортированные версии;
- использовать неактуальные копии как источник.

---

# Реализация

GitHub Repository является мастер-хранилищем GES.

Git история является частью истории развития инженерных записей.
