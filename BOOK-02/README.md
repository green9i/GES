# Книга 2. Управление GES

## Назначение

Книга определяет правила управления GENESIS Engineering Standard.

Она устанавливает:

- правила создания инженерных записей;
- правила идентификации;
- правила версионирования;
- правила изменения;
- правила принятия архитектурных решений;
- правила контроля целостности системы.

---

## Принцип

GES должен развиваться управляемо.

Изменения допускаются только при сохранении:

- истории решений;
- причин изменений;
- взаимосвязей объектов;
- архитектурной целостности.
