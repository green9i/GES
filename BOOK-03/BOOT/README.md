# Workspace Bootstrap Packages

## Назначение

BOOT содержит пакеты инициализации инженерных рабочих пространств GES.

Каждый пакет представляет собой минимальный набор инженерных знаний, необходимый для запуска конкретного рабочего пространства.

---

## Принцип

Рабочее пространство должно получать только тот объем знаний, который необходим для выполнения его инженерной функции.

---

## Состав

- BOOT-00 — Chief Architect
- BOOT-01 — Research Center
- BOOT-07 — Documentation Center

---

## Примечание

BOOT не заменяет Repository.

Repository является Master Source.

BOOT является механизмом синхронизации рабочего пространства с актуальным состоянием Repository.
