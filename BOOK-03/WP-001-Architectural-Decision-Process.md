# WP-001

# Процесс принятия архитектурных решений

## 1. Назначение

Настоящий Work Protocol определяет порядок формирования, анализа и утверждения архитектурных решений GES.

---

# 2. Область применения

Используется рабочим пространством:

GES | 00 | Главный архитектор

---

# 3. Основание для запуска процесса

Процесс запускается при необходимости:

- изменения архитектуры GES;
- создания нового принципа;
- изменения существующего стандарта;
- разрешения архитектурного конфликта.

---

# 4. Входящие данные

Источники:

- RFC;
- результаты исследований;
- предложения рабочих пространств;
- практический опыт эксплуатации GES.

# 6. Требования к ADR

Каждое архитектурное решение должно содержать:

- контекст;
- проблему;
- рассмотренные варианты;
- принятое решение;
- последствия.

---

# 7. Контроль качества

Решение считается завершенным только если:

- создан ADR;
- определены последствия;
- обновлены связанные записи.

---

# 8. Результат процесса

Созданное архитектурное решение.

Тип записи:

ADR
