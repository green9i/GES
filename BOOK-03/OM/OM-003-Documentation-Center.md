# GES | 07 | Documentation Center

## 1. Назначение

Documentation Center является рабочим пространством GES, отвечающим за формализацию инженерных знаний.

---

# 2. Главная функция

Documentation Center переводит:

решения

↓

в инженерные записи GES.

---

# 3. Ответственность

Documentation Center отвечает за:

- оформление инженерных записей;
- соблюдение структуры Markdown;
- контроль шаблонов;
- подготовку мастер-файлов;
- поддержание порядка репозитория.

---

# 4. Полномочия

Documentation Center может:

- проверять оформление;
- выявлять несоответствия стандартам;
- возвращать записи на исправление формы.

---

# 5. Ограничения

Documentation Center не имеет права:

- изменять архитектурный смысл;
- принимать архитектурные решения;
- добавлять новые требования без основания.

---

# 6. Входящие данные

Типы входящих записей:

- ADR;
- RFC;
- ST;
- OM;
- WP;
- WO.

---

# 7. Исходящие результаты

Результаты:

- оформленные инженерные записи;
- структурированные Markdown-файлы;
- подготовленные версии для Git Repository.

---

# 8. Процесс взаимодействия


Архитектурное решение

↓

Documentation Center

↓

Master Record

↓

Git Repository


---

# 9. Главный принцип

Documentation Center отвечает за форму.

Архитектурный владелец отвечает за содержание.
