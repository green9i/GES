# ST-013

# Framework Initialization

## Назначение

Framework Initialization (FI) определяет порядок создания нового рабочего пространства.

---

# Причина создания

Новое рабочее пространство создается только если:

- существует постоянная инженерная функция;
- есть зона ответственности;
- требуется отдельное управление знаниями.

# Запрещено

Создавать рабочие пространства:

- под одну задачу;
- под одного человека;
- без зоны ответственности.
