# Книга 3. Рабочие пространства GES

## Назначение

Книга определяет архитектуру рабочих пространств GENESIS Engineering Standard.

Рабочее пространство является областью ответственности за определенный класс инженерной деятельности.

---

# Принцип

Рабочее пространство отвечает не за документы.

Рабочее пространство отвечает за развитие инженерного знания в своей области.

---

# Отличие от организационной структуры

Рабочее пространство:

- не является подразделением компании;
- не является должностью;
- не является отделом.

Оно является инженерной функцией.
