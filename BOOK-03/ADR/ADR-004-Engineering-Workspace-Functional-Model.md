# ADR-004

# Модель инженерных функций рабочих пространств GES

---

# 1. Context

По мере развития GENESIS Engineering Standard стало необходимо определить универсальную модель инженерных функций.

Рабочие пространства создаются для выполнения инженерной деятельности. Для обеспечения масштабируемости системы необходимо определить единый функциональный подход, независимый от организационной структуры, конкретных исполнителей и программных платформ.

---

# 2. Problem Statement

Требуется определить модель инженерных функций, которая позволит:

- создавать новые рабочие пространства по единым правилам;
- распределять инженерную ответственность;
- заменять исполнителей без изменения архитектуры;
- масштабировать GES без изменения фундаментальных принципов.

---

# 3. Decision

В GES вводится единая модель инженерных функций.

Каждое рабочее пространство определяется набором выполняемых инженерных функций.

Функции определяют область инженерной ответственности и не зависят от конкретного исполнителя, должности или программного обеспечения.

Одно рабочее пространство может выполнять одну или несколько инженерных функций.
