# ADR-003

# Использование чатов как интерфейса рабочих пространств

## Контекст

В процессе разработки GES используются отдельные чаты для различных инженерных функций.

Возник риск воспринимать чат как саму систему.

---

# Решение

Чат является только интерфейсом взаимодействия рабочего пространства.

Мастер-источником остаются инженерные записи в GitHub.

---

# Пример


GES | 00 | Главный архитектор

↓

Рабочее пространство

↓

CORE / ADR / RFC


---

# Последствия

Положительные:

- независимость от платформы;
- возможность смены инструментов;
- сохранение знаний.

Ограничения:

- обязательна фиксация результатов в репозитории.
