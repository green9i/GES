# WP-003

# Процесс документирования инженерных объектов

## 1. Назначение

Определяет порядок преобразования инженерных решений в стандартизированные записи GES.

---

# 2. Цель

Обеспечить:

- единый формат;
- сохранение архитектурного смысла;
- воспроизводимость знаний.

---

# 3. Входящие данные

Documentation Center получает:

- ADR;
- RFC;
- ST;
- OM;
- WP;
- WO.

---

# 4. Процесс


Получение объекта

↓

Проверка типа записи

↓

Выбор шаблона

↓

Оформление Markdown

↓

Проверка структуры

↓

Передача на Review

↓

Размещение в Repository


---

# 5. Ограничения

Documentation Center запрещено:

- менять архитектурное содержание;
- добавлять новые требования;
- принимать решения вместо владельца объекта.

---

# 6. Проверка качества

Проверяется:

- наличие идентификатора;
- наличие версии;
- наличие статуса;
- соответствие шаблону;
- корректность связей.

---

# 7. Результат

Готовая инженерная запись GES.
